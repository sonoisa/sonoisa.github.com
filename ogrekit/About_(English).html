OgreKit is a regular expression framework for Cocoa.
This framework provides functions processing regular expressions equivalent to Ruby and a high-end find panel.

OgreKit uses Oniguruma as its regular expression engine.
Oniguruma is an excellent regular expression library developed by Mr. Kosako. It is used as the regular expression engine of Ruby Language.
